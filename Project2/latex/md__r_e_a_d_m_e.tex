This project generates a {\bfseries{Population}} of random solution vectors and evaluates one selected benchmark objective function using Blind or Repeated Local Search. It outputs the {\bfseries{Fitness}} values (objective values) for each experiment into a CSV file for external analytics. \hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Program utilizes both Blind Search and Local Search algorithms}{Program utilizes both Blind Search and Local Search algorithms}}\label{md__r_e_a_d_m_e_autotoc_md2}
